# Sync Control

Synchronous control navigation items, currently ***only supports one-way synchronization***

```tsx
const SyncCarousel = () => {
    const syncCarouselRef = React.useRef<BearCarousel>(null);

    return <div>
        <BearCarousel
            // ...ignore some
            ref={syncCarouselRef}
        />
        <BearCarousel
            // ...ignore some
            syncCarouselRef={syncCarouselRef}
        />
    </div>
}
```
