# Loop Mode

:::caution Suggestion

Loop Mode may not be supported in some modes
:::

You can use this mode if you want to repeat continuous sliding items

```tsx
<BearCarousel 
    // ...ignore some 
    isEnableLoop
/>
```

You can slide to end then to next
